## 🛠️ Cómo empezar (Instrucciones para alumnos)

1. **Inicia sesión** en tu cuenta de GitHub.
2. Haz clic en el botón verde **[Use this template]** (arriba a la derecha).
3. Selecciona **"Create a new repository"**.
4. En la nueva ventana:
   * **Repository name:** Ponle un nombre claro (ej. `Proyecto-Innovacion-TuNombre`).
   * **Public/Private:** Elige según las indicaciones de tu profesor.
   * **Importante:** Deja **desmarcada** la casilla "Include all branches".
5. Pulsa en **"Create repository from template"**.
